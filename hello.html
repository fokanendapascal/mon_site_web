Bienvenue dans cette formation GIT !

Git est un programme qui permet aux développeurs de conserver un historique des modifications et des versions de tous leurs fichiers.\n Cet outil a trois grandes fonctionnalités:

  • Revenir à une version précédente de votre code en cas de problème
  • Suivre l'évolution de votre code étape par étape
  • Travailler à plusieurs sans risque de supprimer les modifications des autres collaborateurs.
